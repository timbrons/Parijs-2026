# Parijs-reisschema — website

Een mobielvriendelijk, printbaar reisschema voor Parijs (4–7 juli 2026), te hosten via GitHub Pages.

## Bestanden

- **index.html** — de volledige website (één bestand; de Eiffeltoren-afbeelding zit ingesloten).
- **.nojekyll** — leeg bestandje dat voorkomt dat GitHub Pages de pagina onnodig verwerkt.
- **README.md** — dit bestand (hoeft niet mee online, mag wel).

> De lettertypen laden via Google Fonts, dus de pagina heeft bij het bekijken internet nodig. Verder zijn er geen externe bestanden: de afbeelding is in `index.html` ingesloten.

## In 5 stappen online via GitHub Pages

1. **Maak een repository** op GitHub (knop *New*). Geef hem bijvoorbeeld de naam `parijs-reisschema` en zet hem op **Public**.
2. **Upload de bestanden** naar de hoofdmap (root) van de repository: `index.html` en `.nojekyll`. (Via *Add file → Upload files*.)
3. Ga in de repository naar **Settings → Pages**.
4. Kies bij **Source**: *Deploy from a branch*, branch **main** en map **/ (root)**, en klik **Save**.
5. Wacht ~1 minuut. Je site staat dan op:
   `https://<jouw-gebruikersnaam>.github.io/parijs-reisschema/`

## Tips

- Wil je het adres zónder mapnaam (dus `https://<gebruikersnaam>.github.io/`)? Noem de repository dan exact `<jouw-gebruikersnaam>.github.io`.
- Een eigen domein koppelen kan onder **Settings → Pages → Custom domain**.
- Aanpassingen doe je door een nieuwe versie van `index.html` te uploaden; de site werkt zich vanzelf bij.
